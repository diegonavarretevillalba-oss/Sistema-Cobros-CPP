# Sistema de Cobros en C++

## Descripción
Aplicación desarrollada en C++ para gestionar cobros, permitiendo calcular pagos y organizar información de manera estructurada.

## Funcionalidades
- Registro de cobros
- Cálculo de pagos
- Validación de datos de entrada
- Control de flujo mediante menú interactivo

## Tecnologías
- C++
- Programación estructurada
